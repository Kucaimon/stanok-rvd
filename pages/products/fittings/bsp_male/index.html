ФИТИНГ BSP ISO 228-1 Штуцер
Обновлено: 2023/10/03

Гидравлический фитинг BSP ISO 228-1 штуцер представляет собой соединение для РВД с наружной трубной резьбой BSP и уплотнительным конусом 60 градусов. Штуцер предназначен для соединения с элементами системы, имеющими внутреннюю резьбу BSP.

Характеристики фитинга:

  • Стандарт: BSP ISO 228-1, BS 5200, ISO 8434-6
  • Тип: Штуцер (наружная резьба)
  • Резьба: Трубная наружная резьба ISO 228-1 (угол профиля 55°)
  • Уплотнение: Конус 60 градусов
  • Применение: Гидравлические системы с внутренней резьбой BSP
  • Особенности: Соединение с элементами системы через наружную резьбу

Стандарты и спецификации

  • ISO 228-1: Международный стандарт на трубную резьбу
  • BS 5200: Британский стандарт на гидравлические фитинги
  • ISO 8434-6: Стандарт на соединения для гидравлических систем

Применение

Штуцеры BSP применяются в гидравлических системах для соединения РВД с элементами системы, имеющими внутреннюю трубную резьбу BSP. Наружная резьба штуцера вкручивается во внутреннюю резьбу элемента системы, обеспечивая надежное соединение благодаря конусу 60 градусов.

Штуцеры широко используются в промышленном оборудовании, где требуется соединение РВД с гидравлическими блоками, клапанами и другими элементами с внутренней резьбой BSP.
