# 📋 ДЛЯ ПОКАЗА ЗАКАЗЧИКУ

## 🌐 Ссылки на проект

### Вариант 1: GitHub Pages (живой сайт)
Если включен GitHub Pages:
```
https://[ваш-username].github.io/[название-репозитория]/
```

**Как включить GitHub Pages:**
1. Зайдите на GitHub в ваш репозиторий
2. Settings → Pages
3. Source: выберите ветку `main` или `master`
4. Save
5. Через 1-2 минуты сайт будет доступен по ссылке

---

### Вариант 2: Прямая ссылка на репозиторий
```
https://github.com/[ваш-username]/[название-репозитория]
```

Заказчик может:
- Просматривать код
- Скачать проект (Code → Download ZIP)
- Просматривать файлы прямо в браузере

---

### Вариант 3: Показать локально

Если у заказчика установлен локальный сервер:
```bash
# Python
python -m http.server 8000

# Node.js (http-server)
npx http-server

# PHP
php -S localhost:8000
```

Затем открыть: `http://localhost:8000`
