# 🚀 ИНСТРУКЦИЯ ПО РАЗВЕРТЫВАНИЮ

## ✅ Локально готово!

Git репозиторий инициализирован. Все файлы готовы к загрузке.

---

## 📤 ШАГ 1: Создайте репозиторий на GitHub

1. Зайдите на **https://github.com**
2. Нажмите **"+"** → **"New repository"**
3. Название: `rvd-website` (или любое другое)
4. Описание: `Корпоративный сайт-каталог оборудования для производства РВД`
5. **ВАЖНО:** НЕ добавляйте README, .gitignore или лицензию!
6. Нажмите **"Create repository"**

---

## 🔗 ШАГ 2: Подключите удаленный репозиторий

Скопируйте URL вашего нового репозитория и выполните:

```bash
git remote add origin https://github.com/ВАШ-USERNAME/НАЗВАНИЕ-РЕПОЗИТОРИЯ.git
```

Или через SSH (если настроены ключи):
```bash
git remote add origin git@github.com:ВАШ-USERNAME/НАЗВАНИЕ-РЕПОЗИТОРИЯ.git
```

---

## 📤 ШАГ 3: Загрузите код

```bash
git push -u origin main
```

Если будет запрошен логин/пароль:
- Используйте **Personal Access Token** вместо пароля
- Или настройте SSH ключи

---

## 🌐 ШАГ 4: Включите GitHub Pages (опционально)

1. В репозитории: **Settings** → **Pages**
2. **Source:** выберите `main` → **Save**
3. Через 1-2 минуты сайт будет доступен по адресу:
   ```
   https://ВАШ-USERNAME.github.io/НАЗВАНИЕ-РЕПОЗИТОРИЯ/
   ```

---

## ✅ Проверка

```bash
# Проверить подключение
git remote -v

# Проверить статус
git status

# Просмотреть историю
git log --oneline
```
