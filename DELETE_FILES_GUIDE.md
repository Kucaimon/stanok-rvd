# 🗑️ Как удалить файлы из Git репозитория

## Способ 1: Удалить конкретные файлы

```bash
# Удалить файл из Git (но оставить локально)
git rm --cached имя-файла.txt

# Удалить несколько файлов
git rm --cached file1.txt file2.txt file3.txt

# Удалить файл полностью (и из Git, и локально)
git rm имя-файла.txt
```

---

## Способ 2: Удалить папку

```bash
# Удалить папку из Git (рекурсивно)
git rm -r --cached имя-папки/

# Удалить папку полностью
git rm -r имя-папки/
```

---

## Способ 3: Использовать .gitignore

Если хотите игнорировать файлы в будущем:

1. Добавьте файлы/папки в `.gitignore`
2. Затем удалите их из Git:
```bash
git rm --cached файл.txt
```

---

## После удаления:

```bash
# Посмотреть, что будет удалено
git status

# Закоммитить изменения
git commit -m "Удалены лишние файлы"

# Отправить на GitHub
git push
```

---

## Пример: Удалить временные файлы

```bash
# Удалить все .tmp файлы
git rm --cached *.tmp

# Удалить папку node_modules (если случайно добавили)
git rm -r --cached node_modules/

# Закоммитить
git commit -m "Удалены временные файлы"
git push
```

---

## ⚠️ Важно:

- `git rm --cached` - удаляет из Git, но оставляет файл локально
- `git rm` - удаляет и из Git, и локально
- После удаления **обязательно** сделайте `git commit` и `git push`
